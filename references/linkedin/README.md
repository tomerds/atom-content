# LinkedIn References (external)

External, web-sourced fundamentals for writing AI-assisted LinkedIn posts that
don't read as AI slop. This is general craft: the tells that flag AI authorship,
the statistical fingerprints behind them, humanization techniques, and
creator-tested hook frameworks.

**The `linkedin-post` skill reads this folder alongside `playbooks/linkedin/`.**

## How this relates to the playbook

- `playbooks/linkedin/` is **Atom's house voice and rules** (the Stanley playbook).
  It is the source of truth. On any conflict, the playbook wins.
- `references/linkedin/` (this folder) is **general-purpose craft** gathered from
  outside Atom. It backs up and deepens the playbook; it never overrides it.

Where the two overlap (no em dashes, no engagement bait, lead with a stat or
situation, be specific), treat them as reinforcing each other.

## Keeping it current

Each section in `ai-slop-and-fundamentals.md` carries a Sources line. When you add
or revise guidance, cite where it came from so the file stays auditable and easy to
refresh as norms change.
