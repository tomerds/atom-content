# Investor Memo

## Before writing

1. Read every example in `examples/investor-memos/` - match the structure, tone, and level of candor. The examples are the source of truth.
2. Audience is investors, not customers - direct, numbers-forward, honest about lowlights as well as highlights.
3. Never invent or estimate metrics (revenue, customers, pipeline, runway). Use only figures provided by the user or present in the examples; ask for anything missing.

## Process

1. Create a folder at the repo root: `<Month_Year>_Investor_Memo/` (or a topical name).
2. Draft in `draft.md` inside that folder.
3. Flag any claim in the draft you couldn't verify with a `[CONFIRM]` marker so it's easy to review before sending.
